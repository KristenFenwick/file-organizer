# file-organizer

File Organizer (C++).

This repository will stay in **C++**.

## Current status

The organizer is temporarily disabled while parsing is stabilized.

The previous document-keyword parsing approach could create repeated/ambiguous matches, which caused:
- large manual-review output
- duplicate file operations
- harder file recovery

## Direction

- Keep this project focused on a reliable C++ implementation.
- Build a separate repository later for a Python version.
